# 오늘! 밥약? 뭐하지!? - 경희대학교 밥약 루트 플래너

이 웹 애플리케이션은 경희대학교 학우들이 식사 약속(밥약)을 잡을 때 맛집과 카페 코스를 고민 없이 편리하게 계획할 수 있도록 돕는 실시간 보행자 경로 플래너입니다.
경희대학교 주변의 신뢰할 수 있는 상점 정보를 기반으로 사용자의 다양한 선호도에 맞춰 최적의 동선을 설계해 줍니다.

기본적으로 사용자는 지도 마커나 버튼을 통해 SPACE 21관과 정문 중 한 곳을 출발 지점으로 설정할 수 있습니다.
이후 사용자의 선호에 따라 세 가지 방식으로 코스를 탐색합니다.
음식 종류별로 바로 매칭하는 방법, 점심과 저녁 및 심야 등 약속 시간대에 맞춰 영업하는 매장을 매칭하는 방법, 혹은 메뉴와 구체적 위치, 가격대, 식사 시간 등 상세 조건을 인가하여 검색하는 방법을 제공합니다.

식당 매칭이 완료되면 후식으로 방문할 카페 코스를 추가할 수 있습니다.
사용자의 예산 대역(가성비 또는 프리미엄)과 선호 위치에 부합하는 카페 정보를 연계하여 디저트 동선까지 매끄럽게 연결합니다.
만약 메뉴나 예산을 결정하기 어려운 상황이라면 드롭다운 메뉴에 준비된 무작위 선택 기능인 '아몰라골라줘'를 통해 시스템에 랜덤 매칭을 맡길 수 있습니다.

최종적으로 출발지, 추천 식당, 추천 카페, 그리고 돌아갈 복귀 지점을 잇는 최적의 도보 동선을 TMAP 보행자 경로 API를 이용하여 계산합니다.
계산 결과는 실시간으로 지도 위에 시각화되며, 예상되는 총 도보 이동 거리와 예상 소요 시간 요약표를 제공하여 계획적인 밥약 약속 수립을 지원합니다.

* Cartography-Site-Project, 지도학 과제입니다.
